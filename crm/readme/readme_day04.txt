5、列表的全选和取消全选
    1）点击全选按钮，当前页所有活动的复选框的状态随着全选框变化
    2）点击下面的复选框，如果所有的复选框都被选中，全选框同时也要被选中，反之，只要有一个复选框没被选中，全选框页不能被选中
    3）在给复选框绑定单击事件时要注意：
        1》因为活动列表是动态获取的，所以如果使用平常的jQuery绑定事件的方式是不行的，因为当选择器执行的时候，当前活动的列表还没有加载出来，需要从后台获取到才能进行加载，所以不能使用$("#xx").click()这样的方式
        2》给动态生成的元素绑定事件，可以用on函数进行绑定：$("父元素选择器").on("事件名称","要绑定事件的元素",function(){})
            以上on要注意的是：父元素必须是页面的固有元素，不能是动态获取的元素
                            父元素可以是外边很多层的父元素之一，但不能是兄弟元素
                            父元素的范围越小越好，这样找目标元素越好找
    4）如果在当前页中处于全选的状态的情况下进行翻页，会发现翻页后的全选按钮时被选中的状态
        这是由于全选框状态并没有变化，而下面的复选框动态的装载为新的元素
        处理方法：在分页处理函数中的加载完活动列表的后面，将全选框的状态设置为false不选中即可
